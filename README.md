# Hytale Modpack Creator

Herramienta web sencilla para crear modpacks de Hytale con referencias de CurseForge.

## Requisitos
- Python 3 instalado.

## Cómo iniciar

```bash
npm run dev
```

Luego abre [http://localhost:5173](http://localhost:5173) en tu navegador.

## Qué incluye
- Formulario de datos del modpack (nombre, versión, notas).
- Lista de mods con etiquetas y referencia a enlaces de CurseForge.
- Exportación del modpack a JSON para compartirlo.
